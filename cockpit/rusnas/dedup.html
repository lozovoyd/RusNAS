Три уровня дедупликации в rusNAS

Умные копии
Reflinks · Мгновенно · 0 байт
Когда пользователь копирует файл внутри шары, данные на диске не дублируются — оба файла ссылаются на одни и те же блоки
Ночная дедупликация
duperemove · Каждую ночь
Каждую ночь rusNAS сканирует все файлы и объединяет одинаковые блоки данных. Файлы остаются независимыми, но место освобождается
Сетевые копии
vfs_btrfs · SMB · Без трафика
При копировании файлов между папками через SMB сервер выполняет операцию сам, без повторной передачи данных по сети

Частые вопросы

Безопасна ли дедупликация?

Да. rusNAS использует только проверенные методы: данные никогда не удаляются — объединяются только указатели на одинаковые блоки. Если что-то пойдёт не так, файловая система Btrfs автоматически вернёт оригинальные данные.

Влияет ли это на скорость работы?

Нет — для пользователей. Ночная дедупликация запускается в 3:00 с минимальным приоритетом (Nice=19) и не мешает работе. Умные копии работают мгновенно. Сетевая дедупликация ускоряет копирование.

Можно ли отключить дедупликацию для конкретной папки?

Да. На вкладке «Настройки» можно выбрать, для каких томов и шар включена дедупликация.
