# Файловый менеджер (FileBrowser)

FileBrowser -- встроенный веб-файловый менеджер RusNAS. Позволяет просматривать, загружать и скачивать файлы через браузер без необходимости настройки клиентских программ.

---

## Как открыть

FileBrowser доступен по адресу:

```
http://<IP-адрес-RusNAS>/files/
```

Также можно перейти в FileBrowser из дашборда -- нажмите на карточку **"Файловый менеджер"** или ссылку в боковой панели.

!!! note "Примечание"
    Для входа в FileBrowser используются учётные данные, настроенные администратором. Они могут отличаться от системных учётных записей RusNAS.

## Возможности

### Навигация по папкам

- Нажмите на папку для входа в неё
- Используйте хлебные крошки в верхней части для перехода на уровень выше
- Боковая панель показывает дерево каталогов

### Загрузка файлов на сервер

1. Откройте нужную папку
2. Нажмите кнопку **"Загрузить"** в панели инструментов
3. Выберите файлы на вашем компьютере
4. Дождитесь завершения загрузки

Поддерживается загрузка нескольких файлов одновременно и перетаскивание файлов (drag & drop) в область окна.

### Скачивание файлов

1. Выберите файл или папку
2. Нажмите кнопку **"Скачать"** в панели инструментов
3. Папки скачиваются в виде ZIP-архива

### Создание папок

1. Нажмите кнопку **"Новая папка"** в панели инструментов
2. Введите имя папки
3. Нажмите **"Создать"**

### Переименование

1. Выберите файл или папку
2. Нажмите кнопку **"Переименовать"** или нажмите F2
3. Введите новое имя
4. Подтвердите

### Удаление

1. Выберите файлы или папки
2. Нажмите кнопку **"Удалить"** или клавишу Delete
3. Подтвердите удаление

!!! danger "Важно"
    Удаление через FileBrowser является окончательным. Файлы не перемещаются в корзину. Для восстановления используйте [снапшоты](../snapshots/manage.md), если они были настроены.

### Предварительный просмотр

FileBrowser поддерживает предварительный просмотр:

- Изображения (JPG, PNG, GIF, SVG)
- Текстовые файлы
- PDF-документы
- Видео (при поддержке браузером)

Нажмите на файл для открытия предварительного просмотра.

### Общий доступ к файлам

FileBrowser позволяет создавать временные ссылки для общего доступа:

1. Выберите файл
2. Нажмите кнопку **"Поделиться"**
3. Скопируйте сгенерированную ссылку
4. Отправьте ссылку получателю

Получатель сможет скачать файл по ссылке без авторизации.

!!! tip "Совет"
    FileBrowser удобен для быстрого обмена файлами и просмотра содержимого хранилища. Для регулярной работы с большими объёмами данных используйте [SMB/NFS шары](shares.md) -- они обеспечивают лучшую производительность.

## Ограничения

- FileBrowser работает только с файлами на смонтированных томах
- Максимальный размер загрузки зависит от настроек веб-сервера
- Для работы FileBrowser необходимо, чтобы соответствующий сервис был запущен
