# Сертификаты TLS

Управление SSL/TLS-сертификатами для безопасного доступа к rusNAS через HTTPS.

---

## Где найти

Откройте **Сеть** → вкладка **Сертификаты**.

## Установленные сертификаты

Таблица показывает все сертификаты на устройстве:

| Колонка | Описание |
|---------|----------|
| **Домен / Имя** | Доменное имя или Common Name сертификата |
| **Тип** | Let's Encrypt, самоподписанный или импортированный |
| **Издатель** | Удостоверяющий центр (CA) |
| **Действует до** | Дата истечения |
| **Осталось** | Дней до истечения (красный если < 30 дней) |
| **Статус** | Действителен / Истекает / Истёк |

## Получение сертификата Let's Encrypt

Бесплатный доверенный сертификат от Let's Encrypt (автоматическое продление).

### Требования

- Устройство доступно из интернета на порту 80
- Доменное имя указывает на IP устройства (A-запись в DNS)
- certbot установлен (кнопка **Установить certbot** если отсутствует)

### Процесс

1. Нажмите **🔒 Let's Encrypt**
2. Введите доменное имя (например, `nas.company.ru`)
3. Введите email для уведомлений от Let's Encrypt
4. Нажмите **Получить сертификат**
5. Дождитесь завершения проверки домена

!!! tip "DDNS"
    Если у вас динамический IP — настройте DDNS на вкладке [Интерфейсы](interfaces.md) (секция DDNS).

## Самоподписанный сертификат

Подходит для внутренних сетей, где не нужен доверенный CA.

1. Нажмите **+ Самоподписанный**
2. Укажите Common Name (обычно hostname или IP)
3. Выберите срок действия (по умолчанию 365 дней)
4. Нажмите **Создать**

!!! warning "Предупреждение браузера"
    Браузеры показывают предупреждение для самоподписанных сертификатов. Это нормально для внутренней сети.

## Импорт сертификата

Для использования сертификата от корпоративного CA:

1. Нажмите **📥 Импортировать**
2. Загрузите файл сертификата (.crt или .pem)
3. Загрузите приватный ключ (.key)
4. При необходимости загрузите цепочку CA (CA bundle)
5. Нажмите **Импортировать**

## Авто-обновление сертификатов

Демон автоматического обновления проверяет сертификаты и продлевает их за 30 дней до истечения.

| Параметр | Описание |
|----------|----------|
| **Статус** | Активен / Выключен |
| **Последняя проверка** | Дата и время последней проверки |
| **Следующая проверка** | Запланированная проверка |
| **Порог обновления** | 30 дней до истечения |

Кнопка **Показать лог** открывает журнал операций авто-обновления.

## Удаление сертификата

Нажмите кнопку удаления (🗑) в строке сертификата. Удаление нельзя отменить — файлы сертификата будут удалены с диска.

!!! danger "Внимание"
    Удаление активного сертификата приведёт к переключению на HTTP или самоподписанный сертификат.
