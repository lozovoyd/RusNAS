# ИБП: мониторинг и автоотключение

После подключения и настройки ИБП страница мониторинга показывает текущее состояние питания и позволяет настроить автоматическое отключение сервера при разряде батареи.

---

## Панель состояния

На странице **ИБП** отображается панель с ключевыми показателями:

| Показатель | Описание |
|-----------|----------|
| **Заряд батареи** | Текущий уровень заряда в процентах (0-100%) |
| **Оставшееся время** | Расчётное время автономной работы в минутах |
| **Нагрузка** | Текущая нагрузка на ИБП в процентах от максимальной |
| **Входное напряжение** | Напряжение электросети, В |
| **Выходное напряжение** | Напряжение на выходе ИБП, В |
| **Температура** | Внутренняя температура ИБП (если поддерживается) |

## Цветовые индикаторы статуса

Статус ИБП отображается цветным индикатором:

| Цвет | Статус | Описание |
|------|--------|----------|
| Зелёный | **OL** (On Line) | Питание от сети, всё в порядке |
| Жёлтый | **OB** (On Battery) | Работа от батареи, электросеть отключена |
| Красный | **OB LB** (Low Battery) | Батарея разряжается, критический уровень |
| Синий | **CHRG** (Charging) | Батарея заряжается после переключения на сеть |
| Оранжевый | **RB** (Replace Battery) | Батарея изношена, рекомендуется замена |
| Серый | **COMMLOST** | Нет связи с ИБП, проверьте подключение |

## Карточка ИБП на дашборде

На главной странице (Дашборд) карточка ИБП в нижней секции показывает:

- Текущий статус (цветной индикатор)
- Уровень заряда
- Нагрузку

Данные обновляются каждые 30 секунд.

## Настройка автоматического отключения

RusNAS может автоматически завершить работу сервера при критическом разряде батареи, предотвращая потерю данных:

1. На странице **ИБП** найдите раздел **"Авто-отключение"**
2. Настройте параметры:

| Параметр | Описание | Рекомендация |
|----------|----------|--------------|
| **Порог заряда** | При каком уровне заряда начать отключение | 20% |
| **Порог времени** | При каком оставшемся времени автономной работы начать отключение | 5 минут |
| **Задержка** | Время в секундах между сигналом и отключением (для завершения открытых файлов) | 30 секунд |

3. Нажмите **"Сохранить"**

!!! warning "Внимание"
    Убедитесь, что пороги автоотключения установлены с запасом. Сервер должен иметь достаточно времени для корректного завершения всех сервисов и сброса данных на диски.

## Последовательность автоотключения

При срабатывании порогов система выполняет:

1. Отправка уведомления администратору (email/Telegram, если настроены)
2. Закрытие SMB/NFS-сессий
3. Сброс кеша файловой системы на диски
4. Размонтирование томов
5. Корректное завершение работы ОС

## Уведомления

RusNAS может отправлять уведомления при следующих событиях:

| Событие | Описание |
|---------|----------|
| Переход на батарею | Электросеть отключена, ИБП работает от батареи |
| Возврат на сеть | Электричество восстановлено |
| Низкий заряд | Уровень заряда достиг критического порога |
| Замена батареи | ИБП сообщает об износе батареи |
| Потеря связи | Нет связи с ИБП |

## Журнал событий

Для просмотра истории событий ИБП:

1. На странице ИБП нажмите **"Журнал"**
2. Отобразится список событий: переключения питания, ошибки, уведомления

!!! tip "Совет"
    Регулярно проверяйте состояние ИБП. Если индикатор показывает **RB** (Replace Battery), замените батарею как можно скорее -- старая батарея может не обеспечить достаточного времени для корректного отключения.

## Тестирование

Для проверки работоспособности автоотключения:

1. Убедитесь, что все данные сохранены
2. Отключите ИБП от электросети (выдерните кабель питания ИБП)
3. Наблюдайте за страницей мониторинга -- статус должен переключиться на **OB**
4. Подключите ИБП обратно до срабатывания порога автоотключения
