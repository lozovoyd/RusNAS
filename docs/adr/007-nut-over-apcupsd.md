# ADR-007: NUT (Network UPS Tools) вместо apcupsd

## Статус
Принято (2026-03)

## Контекст
Мониторинг и управление ИБП (UPS). Два основных инструмента: apcupsd (APC-specific) и NUT (универсальный).

## Решение
NUT (nut-server + nut-client) версии 2.8.1. Поддерживает сотни моделей UPS через стандартизированные драйверы.

## Последствия
- ✅ Поддержка 300+ моделей UPS (не только APC)
- ✅ Network mode — один NUT server может обслуживать несколько клиентов
- ✅ JSON output (`upsc -j`) в NUT 2.8+ для простого парсинга
- ✅ NOTIFYCMD для интеграции с системой уведомлений
- ⚠️ Конфигурация через 5 файлов в /etc/nut/ (nut.conf, ups.conf, upsd.conf, upsd.users, upsmon.conf)
- ❌ Менее дружелюбная начальная настройка чем apcupsd
